Смехов А.Д. (518/1 или 518_дс)

Задание по курсу "Теория кодирования в криптографии" (И.В. Чижов):
Атака с использованием произведения Адамара (криптосистема Мак-Элиса, коды Рида-Маллера)

Запуск программы: python Rm.py [режим_работы] [аргументы]

Варианты режимов работы:
	1

	По заданным r и m генерируются открытый и закрытый ключи, которые по умолчанию сохраняются в формате двух файлов .txt
	в той же директории, где находится программа.

	Аргументы
		r
		m
		(pk_name = pk.txt) - имя файла открытого ключа (необязательный аргумент)
		(sk_name = sk.txt) - имя файла закрытого ключа (необязательный аргумент)
	Примеры запуска:
		python Rm.py 1 2 5
		python Rm.py 1 2 8 pk1.txt sk1.txt

	2

	По полученному открытому ключу, программа вычисляет секретный ключ и записывает его в формате .txt по умолчанию
	в той же директории, где находится программа. (с помощью атаки с использованием произведения Адамара)

	Аргументы
		(pk_name = pk.txt) - имя файла открытого ключа (необязательный аргумент)
	Примеры запуска:
		python Rm.py 2
		python Rm.py 2 pk1.txt
		
	3

	Программа получает на вход имя файлов открытого и закрытого ключей и выводит на экран true, если ключи соотвествую, 
	false иначе.

	Аргументы
		(pk_name = pk.txt) - имя файла открытого ключа (необязательный аргумент)
		(sk_name = sk.txt) - имя файла закрытого ключа (необязательный аргумент)
	Примеры запуска:
		python Rm.py 3
		python Rm.py 3 pk1.txt sk1.txt

	4

	Программа по полученному открытому ключу и сообщению кодирует сообщение и записывает результат в отдетыльный файл.

	Аргументы
		(pk_name = pk.txt) - имя файла открытого ключа (необязательный аргумент)
		(message_name = message.txt) - имя файла cообщения (необязательный аргумент)
		(en_message = en_message.txt) - имя файла закодированного сообщения (необязательный аргумент)
		(is_random_message = True) - по умолчанию генерирует и сохраняет случайное сообщение, которое в будущем будет закодировано
	Примеры запуска:
		python Rm.py 4
		python Rm.py 4 pk1.txt message.txt en_message.txt 0

	5

	Программа по полученному закрытому ключу и закодированному сообщению кодирует сообщение с помощью алгоритма Рида
	декодирует сообщение.

	Аргументы
		(sk_name = sk.txt) - имя файла закрытого ключа (необязательный аргумент)
		(en_message = en_message.txt) - имя файла закодированного сообщения (необязательный аргумент)
		(de_message = de_message.txt) - имя файла декодированного сообщения (необязательный аргумент)
	Примеры запуска:
		python Rm.py 5
		python Rm.py 5 sk1.txt en_message.txt de_message.txt 0

	6

	Программа по полученным r, m и сообщению декодирует его с помощью алгоритма Рида

	Аргументы
		r
		m
		(message = en_message.txt) - имя файла cообщения (необязательный аргумент)
	Примеры запуска:
		python Rm.py 6 2 5
		python Rm.py 6 2 4 en_message1.txt

Описание формата файлов ключей и сообщения:

Открытый ключ:
	G_pub (каждая строка матрицы в отдельной строке файла, элементы строк матрицы записываются через запятую, без пробелов)

Секретный ключ:
	M^-1 (каждая строка матрицы в отдельной строке файла, элементы строк матрицы записываются через запятую, без пробелов)
	пустая строка
	P^-1 (каждая строка матрицы в отдельной строке файла, элементы строк матрицы записываются через запятую, без пробелов)

Сообщение
	m (элементы сообщения записываются в одну строку через запятую, без пробелов)
