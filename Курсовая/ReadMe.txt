Смехов А.Д. (518/1 или 518_дс)

Запуск программы: python Supp_func.py [Путь к файлу] [Режим работы] [Режим объединения]
Пример: python Supp_func.py test.txt 2 1

[Путь к файлу]:
	Путь к файлу (имя файла) .txt, в котором в первой строке написан вектор значений функции,
	во второй через запятую, без пробелов указаны предполные классы (T0, T1, T2, B, S, L, T01, T02, T12, M0,
	M1, M2, U0, U1, U2, C0, C1, C2)

	Пример содержимого файла:
	012012112
	T0,T1,T2

[Режим работы]:
	Режим работы программы. Существует три типа:
	0 - возращает один из трёх результатов ("Вектор принадлежит классу", "Вектор не принадлежит классу",
		"Вектор может быть дополнен, до принадлежности классу")
	1 - возращает количество векторов, принадлежащих классу, которые можно получить из входного, либо 0,
		если вектор точно не принадлежит классу
	2 - возвращает список векторов, принадлежащих классу, которые можно получить из входного

[Режим объединения]:
	Режим объединения классов. Существует два типа:
	0 - входные классы не объединяются, для каждого класса программа выдаёт свой результат
	1 - входные классы пересекаются, программа выдаёт один результат для объединения классов
	2 - входные классы объединяются, программа выдаёт один результат для объединения классов
